Flujo del Formulario
Pantalla de Inicio (Home)

Bienvenida y Descripción:

En la página de inicio (pages/index.js), tendrás una breve bienvenida al evento y el propósito del formulario: registrar participantes para la Feria Empresarial.

Esto será un mensaje claro y conciso para que los usuarios sepan que deben completar el formulario para registrarse.

Formulario de Registro

El formulario tendrá 6 secciones basadas en las imágenes:

Sección 1: Información Personal

Campos:

Jornada de asistencia: Se elige entre las opciones de jornadas (miércoles/tarde, jueves/tarde, etc.)

Nombres y Apellidos: Campos de texto corto.

Documento de Identidad: Opciones de selección para "Tarjeta de Identidad", "Cédula de ciudadanía", "Cédula de extranjería".

Número de Identidad: Campo de texto corto.

Correo Electrónico: Campo de texto corto.

Número de Teléfono: Campo de texto corto.

Tipo de Asistente: Opciones de "Estudiante", "Acudiente", "Docente", "Directivos", "Administrativos".

Sede Educativa: Opciones como "Principal", "Julio Arboleda", "Emeterio Piedrahita", "Sagrado Corazón".

Sección 2: Estudiante (aplicable si el tipo de asistente es "Estudiante")

Campos:

Grado de Estudio: Campo de texto corto.

Nombre Completo del Estudiante: Campo de texto corto.

Grado del Estudiante: Campo de texto corto.

Sede del Estudiante: Opciones como "Principal", "Sagrado Corazón", "Emeterio Piedrahita", "Julio Arboleda".

Sección 3: Acudiente / Padre de Familia (aplicable si el tipo de asistente es "Acudiente")

Campos:

Parentesco: Campo de texto corto.

Nombre Completo del Estudiante: Campo de texto corto.

Grado del Estudiante: Campo de texto corto.

Sede del Estudiante: Opciones como "Principal", "Sagrado Corazón", "Emeterio Piedrahita", "Julio Arboleda".

Sección 4: Profesores (aplicable si el tipo de asistente es "Docente")

Campos:

Sede Educativa: Opciones como "Principal", "Julio Arboleda", "Emeterio Piedrahita", "Sagrado Corazón".

Sección 5: Declaración de Datos

Política de Privacidad y Manejo de Datos Personales: Texto que informa sobre el tratamiento de los datos personales.

Este texto es importante para cumplir con normativas legales sobre el uso de datos personales y debe ser aceptado por los usuarios.
